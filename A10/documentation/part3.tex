\subsection*{Question}
\begingroup
\begin{verbatim}
Assess the performance of your classifier in each of your categories
by computing precision, recall, and F1.  Note that the definitions
of precisions and recall are slightly different in the context of
classification; see:

http://en.wikipedia.org/wiki/Precision_and_recall#Definition_.28classification_context.29

and

http://en.wikipedia.org/wiki/F1_score
\end{verbatim}
\subsection{Solution}

\begin{enumerate}
\item In order to calculate the Precision , recall and  F1 values TP, TN, FP, FN values are needed. 
\item So I calculate the TP, TN, FP, FN values for each category. The table \ref{table:q3-1} shows these values for each category. 
\item The Precision , Recall , F1 are calculated based on the following 

\[Precesion = \frac{TP}{TP + FP} \]
\[Recall = \frac{TP}{TP + FN} \]
\[F1 = \frac {2TP} {2TP+FP+FN} \]

\item Calculated Precision, Recall and F1 values are shown in the table \ref{table:q3-2}.
